# DigiCar（デジカー）

DigiCar は、ミニ四駆サイズの学習用自動運転ミニカーです。  
プリント基板そのものがシャーシとなる独自構造を採用し、  
RumiCar をリスペクトした教育向けプラットフォームとして開発されています。

2025年8月30日の「デジもく会」にて正式発表されました。

---

## 特徴

- プリント基板がシャーシとして機能する独自構造
- 自動運転機能を搭載予定（ToF センサ対応）
- Bluetooth をメインとした無線通信
- 教育教材としての普及を目指した低コスト設計（想定金額 約5,000円）

---

## システム構成

DigiCar は **電気系・プログラム・機械系** の 3 要素で構成されています。

### ● 電気系
- マイコン：ESP32-WROOM-32E  
- モータードライバ：TB6612FNG（東芝）  
- 電圧レギュレータ：TC1262-3.3VDB05（Microchip）  
- USBシリアル変換：FT231XS（FTDI）  
- 通信方式：Bluetooth（メイン）、Wi-Fi（サブ）

### ● プログラム
- 使用言語：C++（Arduino IDE）

### ● 機械系
- ステアリング：SG-90 サーボモータ  
- 駆動モータ：マブチ FA-130  
- ギア・シャフト・タイヤ：タミヤ製を中心に構成  
- 3Dプリント部品：ギヤケース、ステアリング機構、電池ケース固定部品など

## 開発ストーリー

### 2020年〜：胎動（暗黒時代）
N氏による構想が始まるも、具体化せず停滞。

### 2024年2月：運命の再始動
デジもく会で H氏の「基板、作りませんか？」の一言から再燃。

### 2025年3〜5月：継承と本格始動
京都NTで試作モックを受領し、開発が本格化。

### 2025年8〜9月：公開とコミュニティ化
初発表で手応えを掴み、はんだ付け会や Discord 開設へ。

### 2025年10〜11月：危機と復活
通信不良・異常電流の問題を発見。配線ミスを特定し、改良基板で復活。

### 2026年2月〜現在：次のステージへ
サンプル3台を提供し、マニュアル操作アプリも完成。  
現在は ToF センサ搭載による自動運転へ挑戦中。
