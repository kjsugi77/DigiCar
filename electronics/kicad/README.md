# kicad フォルダについて

このフォルダには、DigiCar の基板設計に使用している KiCad プロジェクトの主要ファイルが含まれています。  
回路図（schematic）と PCB レイアウト（pcb）を編集する際に使用します。

## 含まれるファイル

- **DigiCar.kicad_pro**  
  KiCad プロジェクトファイルです。  
  このファイルを開くことで、回路図エディタと PCB エディタを利用できます。

- **DigiCar.kicad_sch**  
  DigiCar の回路図データです。  
  電気的な接続、部品構成、信号ラインなどを編集できます。

- **DigiCar.kicad_pcb**  
  DigiCar の基板レイアウトデータです。  
  部品配置、配線、基板形状などを編集できます。

## 使い方

1. KiCad を起動し、**DigiCar.kicad_pro** を開きます。  
2. 回路図（Schematic Editor）または基板レイアウト（PCB Editor）を編集できます。


# About the kicad folder

This folder contains the main KiCad project files used for designing the DigiCar PCB.  
These files are used when editing the schematic and PCB layout.

## Included Files

- **DigiCar.kicad_pro**  
  The KiCad project file.  
  Opening this file allows you to access both the schematic editor and the PCB editor.

- **DigiCar.kicad_sch**  
  The schematic file for DigiCar.  
  You can edit electrical connections, component configuration, and signal lines.

- **DigiCar.kicad_pcb**  
  The PCB layout file for DigiCar.  
  You can edit component placement, routing, and the board outline.

## How to Use

1. Launch KiCad and open **DigiCar.kicad_pro**.  
2. Edit the schematic (Schematic Editor) or the PCB layout (PCB Editor) as needed.
