# parts フォルダについて

このフォルダには、DigiCar の部品情報（BOM: Bill of Materials）をまとめたファイルを保存しています。

## DigiCar_BOM_draft.pdf について

**DigiCar_BOM_draft.pdf** は、DigiCar の部品リストとして必要な情報が一通り揃っているものの、  
まだ最終的な確認（誤りの有無、見やすさの調整）が完了していないため、暫定版（draft）として保存しています。

内容の確認と整備が完了次第、正式版の **DigiCar_BOM.csv** として公開する予定です。


# About the parts folder

This folder contains files related to the DigiCar Bill of Materials (BOM).

## About DigiCar_BOM_draft.pdf

**DigiCar_BOM_draft.pdf** includes all the necessary information for the DigiCar parts list.  
However, it is saved as a draft version because the final checks—such as verifying for errors and improving readability—have not yet been completed.

Once the review and refinement are finished, it will be updated and released as the official **DigiCar_BOM.csv**.
