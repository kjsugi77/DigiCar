# DigiCar_AllSet.stl について

**DigiCar_AllSet.stl** は、DigiCar を組み立てるために必要な 3D プリント部品をすべてまとめたデータです。  
すべてのオブジェクトを 1 つの STL ファイルにまとめています。

## 印刷について
- スライサーに読み込んで印刷できます。
- 必要に応じて、スライサー上でパーツを個別に分割して印刷してください。


# About DigiCar_AllSet.stl

**DigiCar_AllSet.stl** contains all the 3D‑printed parts required to assemble the DigiCar.  
All objects are combined into a single STL file.

## Printing Instructions
- Load the file into your slicer and print as needed.
- If necessary, separate the objects in the slicer and print them individually.
